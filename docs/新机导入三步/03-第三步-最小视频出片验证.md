# 第三步：最小视频出片验证

前提：已完成 [01-第一步](./01-第一步-下载必要文件与运行时.md)、[02-第二步](./02-第二步-配置MCP与Skill.md)。  
目标：在 Production Agent 上跑通一条**最短解说片**，导出 `renders/final.mp4`。

本步默认走 **P1 零 Key**（Piper + Remotion）。付费 TTS 为可选加分项，不挡最小验收。

---

## 3.1 开拍前自检

在仓库根、venv 激活、环境变量已设：

```powershell
$env:PYTHONUTF8 = "1"
$env:OPENMONTAGE_PROJECTS_DIR = "<PROJECTS_DIR>"
$env:OPENMONTAGE_P1_ALLOW_WRITES = "true"
$env:PIPER_MODEL_DIR = "<PIPER_MODEL_DIR>"
$env:OPENMONTAGE_PIPER_MODEL = "zh_CN-huayan-medium"
python -c "from openmontage.mcp.doctor.tools import run_doctor; d=run_doctor(); print('can_produce=', d.get('can_produce_video_now')); print(d.get('next_install_for_p1'))"
```

- [ ] `can_produce_video_now == True`  
- [ ] OpenClaw Production Agent 可见 `openmontage-doctor__*` 与 `openmontage-media__*`

---

## 3.2 推荐试跑提示词（零 Key）

对 **Production** Agent 发送（可原样复制）：

> 制作一个约 45 秒动画解说，解释天空为什么是蓝色的；零 Key（Piper），Remotion atelier；每个关卡等我确认后再继续。

期望行为：

1. 走 `animated-explainer` 阶段（research → proposal → script → scene_plan → assets → compose…）  
2. **每个关卡停轮等人审**（Agent 不得替人点通过）  
3. TTS：先 `tts_sample`，你确认试听 OK 后再批量  
4. 合成完成后，成片落在：

```text
<PROJECTS_DIR>/<project_id>/renders/final.mp4
```

- [ ] 文件存在且可用播放器打开  
- [ ] 有画面 + 旁白（时长允许与旁白略有偏差，见下方软验收）

---

## 3.3 人审关卡怎么过（最小路径）

按 Agent 提示逐关确认即可。最小心智模型：

| 关 | 你要确认什么 |
|----|----------------|
| 选题 / proposal | 主题、时长、runtime（Remotion）可接受 |
| script | 旁白文案可接受 |
| scene_plan | 分镜可接受 |
| TTS sample | 试听音色/语速可接受 → 再批量 |
| compose | 允许开始渲染 |
| 成片 | 打开 `final.mp4` 目视/耳听一遍 |

关卡与契约细节见 Skill：`openmontage-production-contract`、`openmontage-gates-intro`。

---

## 3.4 （可选）付费 TTS 替换旁白

仅在你已完成 P2 配置时：

1. 对 Agent 说明要用的商（如 OpenAI）  
2. 强制顺序：`list_tts_providers` → `tts_dry_run` → 你确认估价 → `tts_sample` → 试听 OK → `tts_generate`  
3. 再走 compose

协议见：`openmontage-providers-tts` Skill。  
**不要**在未确认估价/试听时让 Agent 直接批量生成。

## 3.6 常见卡点

| 现象 | 常见原因 |
|------|----------|
| MCP 起不来 | `cwd` 不是仓库根；用了系统 Python 而非 venv |
| 写盘失败 | 未设 `OPENMONTAGE_P1_ALLOW_WRITES=true` |
| `can_produce_video_now=false` | 缺 Piper 模型 / Remotion npm / FFmpeg |
| Agent 乱跳阶段 | 未启用 `openmontage-animated-explainer` |
| 成片无中文 | 缺中文字体 |
| 渲染很慢 | CPU 无 GPU，属正常；可先缩时长做冒烟 |
