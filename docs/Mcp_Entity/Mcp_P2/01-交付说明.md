## 1. 产品承诺

在 P1 零 Key 出片之上，可选启用云端 TTS，获得更自然旁白。必须：

- 先试听后批量

- Key 只在环境变量

## 2. 交付内容

```text
openmontage/mcp/providers_tts/     # python -m openmontage.mcp.providers_tts
openmontage/skills/openmontage-providers-tts/SKILL.md
tests/mcp_p2/
```

### MCP 工具

| Tool | 作用 |
|------|------|
| `list_tts_providers` | 列出 openai/elevenlabs/dashscope/doubao/google/kling |
| `tts_dry_run` | 估价，不生成 |
| `tts_sample` | 试听；需 `confirm_estimate=true` |
| `tts_generate` | 批量；需 `confirm=true` 且 `confirm_sample_ok=true` |

### Provider → BaseTool

| provider | tool | 典型 Key |
|----------|------|----------|
| openai | openai_tts | OPENAI_API_KEY |
| elevenlabs | elevenlabs_tts | ELEVENLABS_API_KEY |
| dashscope | dashscope_tts | DASHSCOPE_API_KEY |
| doubao | doubao_tts | DOUBAO_SPEECH_API_KEY |
| google | google_tts | GOOGLE_API_KEY / GEMINI_API_KEY |
| kling | kling_tts | KLING_API_KEY |

### 环境闸门

| 变量 | 作用 |
|------|------|
| `OPENMONTAGE_ALLOWED_PROVIDERS` | 逗号白名单，如 `openai,dashscope` |
| `OPENMONTAGE_MAX_COST_USD` | 估价超限则拒绝 sample/generate |
