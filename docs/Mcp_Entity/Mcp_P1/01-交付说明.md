# P1 交付说明

> 日期：2026-07-17  
> 状态：已实现（代码 + 本目录文档）

## 1. 产品承诺

在 **无付费云 Key** 环境下，通过 OpenClaw Agent + Skills 驱动，走完 `animated-explainer`：

```text
research → proposal(人审) → script(人审)
→ scene_plan(人审) → assets(试听/人审)
→ edit → compose → final.mp4
```

不提供 `make_video()`；Agent 为脑，MCP 为手，Python BaseTool 复用。

## 2. 双 Agent 策略

| Agent | 写盘 | MCP | Skills |
|-------|------|-----|--------|
| **Default**（诊断） | 否 | doctor 只读子集 | router + gates-intro |
| **Production**（出片） | `OPENMONTAGE_P1_ALLOW_WRITES=true`，仅沙箱 | doctor 全量写 API + media | + explainer + production-contract + L3 指针 |

所有路径必须落在 `OPENMONTAGE_PROJECTS_DIR`。

## 3. 交付内容

### 代码

```text
openmontage/
  mcp/
    doctor/          # + read/write_artifact, checkpoint, approve, decision, init_project(门控)
    media/           # python -m openmontage.mcp.media
    common/          # + jobs.py, registry.py
  skills/
    openmontage-animated-explainer/
    openmontage-production-contract/
    openmontage-l3-remotion|tts|ffmpeg/
tests/mcp_p0/        # 回归
tests/mcp_p1/        # P1 单元测试
```

### MCP-B：`openmontage-media`

`list_media_tools` · `tts_preflight` · `tts_sample` · `tts_generate` · `generate_diagram` · `generate_subtitles` · `compose_preflight` · `compose_start` · `job_status` · `job_cancel` · `probe_media` · `mix_audio` · `stitch_video`

### MCP-A 增量（doctor）

`read_artifact` · `write_artifact` · `write_checkpoint` · `approve_checkpoint` · `append_decision` · `init_project`（需 P1 write flag）

### Skills

- `openmontage-animated-explainer` — 阶段循环 + director 指针
- `openmontage-production-contract` — Rule Zero / 关卡 / runtime
- L3 指针：`openmontage-l3-remotion` / `openmontage-l3-tts` / `openmontage-l3-ffmpeg`
