## 2. 交付内容

### 代码

```text
openmontage/
  mcp/
    doctor/          # python -m openmontage.mcp.doctor
    common/          # envelope / sandbox / errors
  skills/
    openmontage-router/SKILL.md
    openmontage-gates-intro/SKILL.md
tests/mcp_p0/        # 单元测试
```

### 默认暴露的 MCP 工具（只读）

`doctor` · `provider_menu_summary` · `list_pipelines` · `list_projects` · `get_project_state` · `get_next_stage` · `validate_artifact` · `validate_checkpoint` · `estimate_cost`

`init_project` 已注册；**无** `OPENMONTAGE_P1_ALLOW_WRITES` 时拒绝写盘。P1 Production Agent 可开启沙箱内写盘（见 [../Mcp_P1/00-INDEX.md](../Mcp_P1/00-INDEX.md)）。

### 不做（P0 范围）

- 视频/TTS 生成、`make_video`、providers MCP（P1 media / P2 providers）

## 3. 依赖

- Python 3.10+
- `mcp>=1.0`（已写入 `requirements.txt`）
- 运行时 `cwd` / `PYTHONPATH` 指向仓库根，以便 `import tools` / `import lib`

## 4. 快速自检

```powershell
cd <OPENMONTAGE_REPO_ROOT>
.\.venv\Scripts\Activate.ps1
$env:PYTHONUTF8 = "1"
$env:OPENMONTAGE_PROJECTS_DIR = "$PWD\projects"
python -c "from openmontage.mcp.doctor.tools import run_doctor; import json; print(json.dumps(run_doctor(), ensure_ascii=False, indent=2)[:2000])"
python -m pytest tests/mcp_p0 -q
```
