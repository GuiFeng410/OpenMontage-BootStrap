# P0 验收清单

## 自动化

- [ ] `python -m pytest tests/mcp_p0 -q` 通过
- [ ] `run_doctor()` 返回 `contract_version` / `can_produce_video_now=false`
- [ ] 路径 `../` 逃逸被 `SandboxError` 拒绝
- [ ] `init_project` 工具返回策略错误（不写盘）
- [ ] `openclaw mcp probe openmontage-doctor`（有 OpenClaw 时）只见只读工具白名单

## 对话（OpenClaw）

- [ ] 「我想做个视频」→ 能力摘要 + **恰好 3** 条提示词
- [ ] 无付费 Key 时不把付费路径写成唯一选项
- [ ] 未放行 `bundle-mcp` 时的排查步骤可复现（见安装文档）
- [ ] 新人能复述：下一步装什么、将来成片在 `$OPENMONTAGE_PROJECTS_DIR/<id>/renders/final.mp4`

## 非回归

- [ ] 无 `make_video`
- [ ] Skill 内无 API Key
- [ ] 未改 Rule Zero / pipeline 语义
