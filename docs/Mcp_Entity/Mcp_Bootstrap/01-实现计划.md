## 6. 工具清单

### 6.1 bootstrap_*（Skill01）

| 工具 | 说明 | 门禁 |
|------|------|------|
| `clone_repo` | 可选：仓已存在时可跳过；支持 github/gitee/url/auto | 覆盖目录需 confirm |
| `detect_environment` | 包装 `run_doctor` | 无 |
| `plan_install` | **只生成计划**（等价总 dry_run） | 无 |
| `install_python_deps` | venv + pip | 默认 dry_run；执行需 confirm |
| `install_node_deps` | remotion-composer npm | 同上 |
| `ensure_ffmpeg` | 检测；尝试安装或返回命令 | 改 PATH/系统包需 confirm |
| `ensure_piper_model` | 下载中文模型到指定目录 | 下载需 confirm |
| `configure_sandbox` | 创建 projects 目录、返回推荐 env | 无 |
| `verify_ready` | `can_produce_video_now` + 缺口 | 无 |

**约定：** 所有会改机器状态的工具：`dry_run: bool = True`；`dry_run=False` 时必须 `confirm_execute=true`。

### 6.2 produce_*（Skill02，最小集）

| 工具 | 内部复用 |

| `produce_read_state` / `produce_get_next_stage` | doctor |
| `produce_tts_preflight` / `produce_tts_sample` / `produce_tts_generate` | media Piper（保留试听门禁） |
| `produce_generate_subtitles` | media（解说常用） |
| `produce_compose_preflight` / `produce_compose_start` / `produce_job_status` | media Remotion |

明确首版 **不暴露**：diagram、stitch、mix、providers_tts。
