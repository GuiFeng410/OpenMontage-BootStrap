## 交付内容

| 项 | 路径 |
|----|------|
| MCP | `openmontage/mcp/bootstrap/` → `python -m openmontage.mcp.bootstrap` |
| Skill01 | `openmontage/skills/openmontage-bootstrap-setup/` |
| Skill02 | `openmontage/skills/openmontage-bootstrap-produce/` |
| 安装 Skill（可单独拷到 OpenClaw） | `openmontage/skills/openmontage-bootstrap-installer/` |
| 测试 | `tests/mcp_bootstrap/` |
| 模板 | [templates/](./templates/) |

## 用户最小路径

### 路径 A（推荐新机）

1. 将 `openmontage-bootstrap-installer` **复制**到 OpenClaw 用户 Skill 目录并启用  
2. 用户说「我想生成视频」→ 安装 Skill 引导 clone（只口述配 MCP，不改配置文件）  
3. 启用仓内 setup + produce → Skill01 装环境 → Skill02 出片  

### 路径 B（已手动 clone）

1. 手动 clone（GitHub 失败用 Gitee）  
2. OpenClaw 只注册本门面 MCP + setup/produce  
3. Skill01：`plan_install` → 用户确认 → 带 `confirm_execute` 执行 → `verify_ready`  
4. Skill02：零 Key 最简解说出片  

操作说明：[../../README/00-INDEX.md](../../README/00-INDEX.md)

- 无管理员：返回 `manual_commands`，不硬崩  
- produce 最小集；无 diagram/stitch  
- 内部复用 doctor/media
