# OpenMontage BootStrap（中文入口）

开源 **Agent 视频制作**。BootStrap 路径：可选先把 **安装 Skill** 拷进 OpenClaw → clone 本仓 → **1 个门面 MCP** + setup/produce → 零 Key 最小解说成片。

| 优先级 | 平台 | URL |
|--------|------|-----|
| 1 | GitHub | https://github.com/GuiFeng410/OpenMontage-BootStrap |
| 2 | Gitee（备用） | https://gitee.com/rory_-3232/open-montage-boot-strap |
