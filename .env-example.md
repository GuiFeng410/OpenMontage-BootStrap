# OpenMontage 环境变量配置（分类精简白话版）

**使用方法：复制全部内容保存为 .env 文件，按需填写对应 API 密钥即可，未使用的功能可直接留空**

---

## **【一、全能型 AI 服务（图文音视频全能力）】**

**GOOGLE_API_KEY=**              # 谷歌全能密钥，支持绘图、多语种配音、视频生成/编辑（付费解锁完整版）                                  # 申领地址：++[https://aistudio++.google.com/apikey](https://aistudio.google.com/apikey)

# **GEMINI_API_KEY=**            # GOOGLE_API_KEY别名，同时填写时此密钥优先级更高

**GOOGLE_APPLICATION_CREDENTIALS=**   # 谷歌云服务账号JSON密钥路径（备用登录方式）

**GOOGLE_CLOUD_PROJECT=**             # 谷歌云项目ID（调用谷歌绘图功能必填）

**GOOGLE_CLOUD_LOCATION=**            # 谷歌云服务区域，默认美国中部 us-central1

**DASHSCOPE_API_KEY=**           # 阿里通义千问全能密钥，支持绘图、配音、语音转文字                                  # 申领地址：++[https://dashscope++.aliyun.com/](https://dashscope.aliyun.com/)

## **【二、视频生成专项服务】**

**FAL_KEY=**                     # 核心视频/绘图网关，支持绝大多数AI视频、图像模型                                  # 申领地址：++[https://fal++.ai/dashboard/keys](https://fal.ai/dashboard/keys)

**FAL_AI_API_KEY=**              # FAL_KEY备用别名，填任意一个都能生效

**REPLICATE_API_TOKEN=**         # 备用视频通道，专门调用Seedance视频模型                                  # 申领地址：++[https://replicate++.com/account/api-tokens](https://replicate.com/account/api-tokens)

**KLING_API_KEY=**               # 可灵官方密钥，主打高清视频、数字人、唇形同步

**KLING_API_BASE_URL=**          # 可灵接口地址（可选）                                  # 海外默认：++[https://api++-singapore.klingai.com](https://api-singapore.klingai.com)                                  # 国内稳定：++[https://api++-beijing.klingai.com](https://api-beijing.klingai.com)

**HIGGSFIELD_API_KEY=**          # Higgsfield云视频服务密钥

**HIGGSFIELD_API_SECRET=**       # Higgsfield云视频服务秘钥

# **HIGGSFIELD_KEY=**            # 合并密钥（二选一）格式：密钥:秘钥，可替代上面两个参数

**HEYGEN_API_KEY=**              # 一站式聚合密钥，通用多款主流视频模型

**RUNWAY_API_KEY=**              # Runway Gen-4 高端视频生成专用密钥

**VIDEO_GEN_LOCAL_ENABLED=**     # 是否开启本地视频生成，true开启（需独立GPU）

**VIDEO_GEN_LOCAL_MODEL=**       # 本地视频模型选型：wan2.1-1.3b、wan2.1-14b、hunyuan-1.5等

**MODAL_LTX2_ENDPOINT_URL=**    # 自建LTX2视频模型接口（非必填）
