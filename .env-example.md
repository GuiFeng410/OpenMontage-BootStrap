# OpenMontage 环境变量配置（分类精简版）

## 怎么用

1. 复制下文「可粘贴到 .env」各节中的 `变量=` 行 → 另存为仓库根目录 `.env`（或写入 OpenClaw 对应 MCP 的 `env`）
2. 先查文首总表：按「听说过的模型名」找到该填哪一行；**申领地址在各节变量旁**，不塞进总表
3. 不用的能力**留空**即可；空 Key 时系统不会调用该能力

---

## 文首总表 A｜模型 / 能力 → 填哪一栏（反向索引）


| 渠道                                | 填哪个变量                                | 见第几节   |
| --------------------------------- | ------------------------------------ | ------ |
| **Agnes**（图+视频，省钱优先）              | `**AGNES_API_KEY`**                  | **二**  |
| 阿里 / 通义 / 百炼 / DashScope（图+配音+转写） | `DASHSCOPE_API_KEY`                  | 一      |
| 谷歌全能（图+多语配音+Veo 等）                | `GOOGLE_API_KEY`（或 `GEMINI_API_KEY`） | 一      |
| **Seedance**（国内常听说的 AI 视频）        | `**FAL_KEY`**                        | **二**  |
| **MiniMax** 视频                    | `**FAL_KEY`**                        | **二**  |
| Veo                               | `FAL_KEY` **或** `GOOGLE_API_KEY`     | 二 / 一  |
| Kling / 可灵（官方直连）                  | `KLING_API_KEY`                      | 二      |
| Sora                              | `OPENAI_API_KEY`                     | 三（兼视频） |
| Runway                            | `RUNWAY_API_KEY`                     | 二      |
| Seedance 备用通道                     | `REPLICATE_API_TOKEN`                | 二      |
| Higgsfield 云视频                    | `HIGGSFIELD_API_KEY` + `_SECRET`     | 二      |
| FLUX 等生图（经 FAL 网关）                | `FAL_KEY`                            | 二      |
| ElevenLabs 配音 / 音效 / 乐            | `ELEVENLABS_API_KEY`                 | 三 / 四  |
| 豆包中文配音                            | `DOUBAO_SPEECH_API_KEY`              | 三      |
| Pexels / Pixabay 免费素材             | 各自 Key                               | 五      |
| Piper 本地配音                        | **无需 Key**                           | 三      |


**注意点：** 很多视频模型**没有单独一行**，因为共用「网关 Key」（尤其 FAL）。不会 FAL 没关系——要用 Seedance / MiniMax → 填 `FAL_KEY`。有 Agnes 时优先填 `AGNES_API_KEY`（图+视频同一 Key）。

## 文首总表 B｜初次省钱（建议）


| 建议               | 做法                                                                  |
| ---------------- | ------------------------------------------------------------------- |
| 先试 Agnes（图+视频）   | 只填 `AGNES_API_KEY` → 省钱真·AI 图/视频；选择器在有 Key 时会自动首选 Agnes             |
| 先试阿里、缓开视频        | 只填 `DASHSCOPE_API_KEY` → 付费图+配音；画面可先轻度模板 / 中度 Stock                 |
| 真要其它 AI 视频再开 FAL | 需要 Seedance 等时再填 `FAL_KEY`，不要一上来全开                                  |
| 图模型选便宜档          | DashScope 优先试 `z-image-turbo` / `wan2.7-image`，少用 `*-max` / `*-pro` |
| 视频先短样片           | 门禁：估价 → **约 4～5 秒 sample** → 确认后再正式生成                               |
| 设预算上限            | 加：`OPENMONTAGE_MAX_COST_USD=2`（可按钱包改）                               |
| 白名单收窄            | 例：`OPENMONTAGE_ALLOWED_PROVIDERS=agnes`；或 `agnes,dashscope`         |
| 配音可先免费           | Piper 零 Key；确认文案后再换 DashScope / 其它付费 TTS                            |


可选（写入 `.env` 或 MCP env）：

```text
OPENMONTAGE_MAX_COST_USD=2
OPENMONTAGE_ALLOWED_PROVIDERS=agnes
```

## 文首总表 C｜出片三档最少填什么


| 档         | 最少要填                                                                 |
| --------- | -------------------------------------------------------------------- |
| 轻度        | 可不填付费 Key（Piper + 模板）                                                |
| 中度        | 建议第五节 Pexels/Pixabay；配音仍可 Piper                                      |
| 重度图+配音（省） | `DASHSCOPE_API_KEY` 或 `AGNES_API_KEY` + 上表省钱建议                       |
| 重度真·AI 视频 | 上项 + `**AGNES_API_KEY`（优先）或 FAL_KEY / Kling / 谷歌 / OpenAI 等**（按总表 A） |

# 可粘贴到 .env（一～七）

> **申领地址约定：** 写在对应 `变量=` 下一行的 `# 申领：` 注释里；填 Key 时顺便点开。

## 【一、全能型 AI 服务】

**备注：**

- **填一个 Key 即可**解锁该商在本项目里已接通的能力（不必每个子能力各填一次）
- **不等于「所有视频模型」**：阿里当前 = 生图 + TTS + 语音转写；**不能单靠它做 Seedance / Kling / Sora 真·文生视频**
- 谷歌 Key 覆盖面更广（含 Veo 等路径），仍以总表 A 为准

```env
GOOGLE_API_KEY=              # 谷歌全能：绘图、多语种配音、视频生成/编辑（付费解锁完整能力）
# 申领：https://aistudio.google.com/apikey

# GEMINI_API_KEY=            # GOOGLE_API_KEY 别名；两都填时此项优先

GOOGLE_APPLICATION_CREDENTIALS=   # 谷歌云服务账号 JSON 路径（备用登录，一般可空）
GOOGLE_CLOUD_PROJECT=             # 谷歌云项目 ID（走 Vertex 绘图时必填）
GOOGLE_CLOUD_LOCATION=            # 区域，默认 us-central1

DASHSCOPE_API_KEY=              # 阿里通义/百炼一个 Key：绘图 + 配音 + 语音转文字（不含真·文生视频）
# 申领：https://dashscope.aliyun.com/
```

---



## 【二、视频生成专项服务】

**备注：**

- `**AGNES_API_KEY` = Agnes 图+视频**（省钱优先；选择器有 Key 时自动首选）
- `**FAL_KEY` = 视频/绘图网关**，不是又一个神秘品牌
- 国内常听的 **Seedance、MiniMax**（以及部分经 FAL 的 Veo / FLUX 图）→ **填 FAL 这一栏**
- Kling 官方、Runway、HeyGen、Higgsfield、Replicate 等有独立 Key 的，填各自行

```env
AGNES_API_KEY=               # Agnes 图（2.0/2.1 Flash）+ 视频（Video V2.0）；省钱优先
# 申领：https://agnes-ai.com/ （控制台 API Key）
# AGNES_AI_API_KEY=          # AGNES_API_KEY 别名，填任意一个即可
# AGNES_ACCOUNT_TIER=default # default|enterprise|tokenplan（视频默认并发 1/2/3）
# AGNES_VIDEO_MAX_CONCURRENCY=  # 可选并发覆盖；仍受档位上限约束

FAL_KEY=                     # 核心网关：Seedance / MiniMax / 多数 AI 视频与部分生图；不会 FAL 也填这里
# 申领：https://fal.ai/dashboard/keys

FAL_AI_API_KEY=              # FAL_KEY 备用别名，填任意一个即可

REPLICATE_API_TOKEN=         # 备用视频通道（如 Seedance 走 Replicate）
# 申领：https://replicate.com/account/api-tokens

KLING_API_KEY=               # 可灵官方：高清视频、数字人、唇形同步等
# 申领：可灵开放平台控制台（按你账号地区）

KLING_API_BASE_URL=          # 可灵接口地址（可选，一般留空）
# 海外默认：https://api-singapore.klingai.com
# 国内稳定：https://api-beijing.klingai.com

HIGGSFIELD_API_KEY=          # Higgsfield 云视频密钥
HIGGSFIELD_API_SECRET=       # Higgsfield 云视频秘钥
# HIGGSFIELD_KEY=            # 合并写法（二选一）：密钥:秘钥，可替代上面两个

HEYGEN_API_KEY=              # 一站式聚合：多款主流视频模型
RUNWAY_API_KEY=              # Runway Gen-4 直连

VIDEO_GEN_LOCAL_ENABLED=     # 本地视频生成：true 开启（需独立 GPU）
VIDEO_GEN_LOCAL_MODEL=       # 本地模型：wan2.1-1.3b、wan2.1-14b、hunyuan-1.5 等
MODAL_LTX2_ENDPOINT_URL=     # 自建 LTX2 接口（非必填）
```

## 【三、语音配音专项服务】

**备注：** Piper 本地配音**无需密钥**（安装依赖即可）。`OPENAI_API_KEY` 兼 **Sora** 视频（见总表 A）。

```env
ELEVENLABS_API_KEY=          # 高品质 AI 人声配音、音效、背景音乐相关
# 申领：https://elevenlabs.io/（控制台 API Key）

OPENAI_API_KEY=              # 备用配音；同时支持 AI 绘图；兼 Sora 视频
# 申领：https://platform.openai.com/api-keys

XAI_API_KEY=                 # Grok：绘图、修图、视频、配音
# 申领：https://console.x.ai/

DOUBAO_SPEECH_API_KEY=       # 火山豆包中文优质配音
DOUBAO_SPEECH_VOICE_TYPE=    # 默认音色，示例：zh_female_vv_uranus_bigtts
# 申领：火山引擎语音控制台
```

## 【四、音乐生成专项服务】

**备注：** 零 Key 可先用本地 BGM / 无乐出片；需要专业 AI 作曲再填。

```env
SUNO_API_KEY=                # 专业 AI 音乐：完整歌曲、纯音乐、多曲风
# 申领：Suno 开放平台 / 你所用的代理控制台
```

## 【五、免费图片 / 视频素材资源】

**备注：** 中度档「实拍素材」常用；**不是**文生视频。免费但有额度/限速。

```env
PEXELS_API_KEY=              # 免费无版权高清视频、图片
# 申领：https://www.pexels.com/api/

PIXABAY_API_KEY=             # 免费无版权高清视频、图片
# 申领：https://pixabay.com/api/docs/

UNSPLASH_ACCESS_KEY=         # 免费无版权高清图片
# 申领：https://unsplash.com/developers
```

## 【六、语音识别 & 音频分析】

**备注：** 字幕精细对齐、多人说话区分等再填；日常出片可先空。

```env
HF_TOKEN=                    # 语音转文字相关（如说话人区分）
# 申领：https://huggingface.co/settings/tokens

AZURE_SPEECH_KEY=            # Azure 云端高精度语音识别
AZURE_SPEECH_REGION=         # 区域，示例：eastus
# AZURE_SPEECH_ENDPOINT=     # 自定义接口地址（非必填）
# 申领：Azure 门户 → Speech 资源 → Keys and Endpoint
```

## 【七、本地部署专属功能（无需云端密钥）】

**备注：** 需本机自行克隆/安装对应工程；默认可全部留空或保持注释。

## 填完之后

1. 仓库：保存为 `.env`（勿提交真实 Key 到 git）
2. OpenClaw：把同一 Key 写入对应 MCP 的 `env` 并**重启**该 MCP
  - 图 → `openmontage-providers-image`  
  - 配音 → `openmontage-providers-tts`  
  - 真·AI 视频 → `openmontage-providers-video`
3. 启用对应执行 Skill；出片走门禁：list → dry_run → sample → generate
