# 配置 — 字段清单

> 只列名称与模板路径，**不含操作步骤**。步骤见 [说明/01](../说明/01-安装配置与环境.md) 与对应 Skill。  
> JSON 模板目录：[templates/](./templates/)

## MCP Server ↔ 模板 ↔ 模块

| Server 名 | 模板文件 | `args` 模块 |
|-----------|----------|-------------|
| `openmontage-bootstrap` | `templates/bootstrap.mcp.json` | `openmontage.mcp.bootstrap` |
| `openmontage-providers-tts` | `templates/providers-tts.mcp.json` | `openmontage.mcp.providers_tts` |
| `openmontage-providers-image` | `templates/providers-image.mcp.json` | `openmontage.mcp.providers_image` |
| `openmontage-providers-video` | `templates/providers-video.mcp.json` | `openmontage.mcp.providers_video` |
| `openmontage-providers-stock`（可选） | `templates/providers-stock.mcp.json` | `openmontage.mcp.providers_stock` |

## 通用字段

| 字段 | 典型值 |
|------|--------|
| `command` | `<REPO>\.venv\Scripts\python.exe`（推荐绝对路径） |
| `args` | `["-m", "openmontage.mcp.<模块>"]` |
| `cwd` | `<REPO>` |
| `env.OPENMONTAGE_PROJECTS_DIR` | 各 server 同一沙箱目录 |
| `env.PYTHONUTF8` | `1` |

## 门面额外 env

| 变量 | 值 |
|------|-----|
| `OPENMONTAGE_P1_ALLOW_WRITES` | `true` |

## 付费 Key（providers-tts / image / video）

| Key | 常见用途 |
|-----|----------|
| `FAL_KEY` | flux 图；seedance / minimax 视频 |
| `OPENAI_API_KEY` | openai TTS/图；sora |
| `DASHSCOPE_API_KEY` | dashscope TTS/图 |
| `KLING_API_KEY` | kling TTS；官方 Kling 图/视频 |
| `GEMINI_API_KEY` / `GOOGLE_API_KEY` | google TTS/图；veo |
| `XAI_API_KEY` | grok 图 |
| `RUNWAY_API_KEY` | runway 视频 |
| `ELEVENLABS_API_KEY` | ElevenLabs TTS |
| `DOUBAO_SPEECH_API_KEY` | 豆包 TTS |
| `OPENMONTAGE_MAX_COST_USD` | 估价上限（可选） |
| `OPENMONTAGE_ALLOWED_PROVIDERS` | 白名单，如 `flux,kling`（可选） |

## Stock（可选）env

| 变量 | 说明 |
|------|------|
| `PEXELS_API_KEY` | 免费申请 |
| `PIXABAY_API_KEY` | 免费申请 |
| `OPENMONTAGE_ALLOWED_STOCK_SOURCES` | 如 `pexels,pixabay` |
