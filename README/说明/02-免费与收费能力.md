# 02 — 免费与收费能力

> 出片编排：`openmontage-bootstrap-produce`（主题确认后选档）。  
> 字段/模板：[../配置/00-字段清单.md](../配置/00-字段清单.md)

## 出片三档（怎么选）

| 档位 | 费用 | 画面 | 语音 | 典型准备 |
|------|------|------|------|----------|
| **轻度** | $0 | 模板/字幕 | **Edge-TTS 男声**（需联网；默认 `zh-CN-YunyangNeural`） | setup 通过 + `edge-tts` |
| **中度** | Stock $0；TTS 可选付费 | Stock 搜图/搜视频 | 默认同上 Edge；可**显式**改付费 TTS | Stock MCP 已默认注册；填免费 Key 后启用 |
| **重度** | 按云厂商 | 付费生图 + 生视频 | 付费高级 TTS | 三个付费 MCP 已注册并填 Key |

改档须你明确说「升到中度 / 升到重度 / 改回轻度」。档位写入 `project.json` → `production_profile`。

## 免费侧

### 配置是什么

- 门面 MCP `openmontage-bootstrap`（含 `OPENMONTAGE_P1_ALLOW_WRITES`）  
- 本地：venv、ffmpeg、`edge-tts`（中文旁白默认）、Piper 模型（离线回退，由 setup 检测安装）  
- **不必**填付费云 Key  
- **中度画面**：默认已注册 MCP `openmontage-providers-stock`；填 Pexels/Pixabay **免费** Key 后才调用（见 `.env-example.md`）

### 选择有哪些

| 选择 | 说明 |
|------|------|
| 轻度出片 | 首推；Edge 男声（按字幕 cue 对齐）+ compose → `final.mp4`；无网再用 Piper |
| 中度 + Stock | search → 确认 → `download(confirm, project_id=…)` 写 `asset_manifest` |
| 字幕 + BGM | 见 [03-字幕与配乐.md](./03-字幕与配乐.md)；默认零 Key |

### 对应 Skill / MCP

| 能力 | Skill | MCP |
|------|--------|-----|
| 环境 | `openmontage-bootstrap-setup` | 门面 |
| 出片编排 | `openmontage-bootstrap-produce` | 门面 `produce_*` |
| 成片简报确认 | `production_to_usercheck` | —（确认后交接 produce） |
| Stock | `openmontage-providers-stock` | `openmontage-providers-stock` |
| 字幕配乐 | `openmontage-bootstrap-captions-music` | 门面 `produce_*` 文稿/音乐工具 |

轻度示例口令：

> 制作约 45 秒动画解说……；选轻度（Edge 男声旁白）；每个关卡等我确认。

## 收费侧

### 配置是什么

- 安装时已注册：`openmontage-providers-tts` / `image` / `video`（Key 可空）  
- 用时在对应 MCP **env 填 Key** 后重启；不必重配 command/cwd  
- 可选：`OPENMONTAGE_MAX_COST_USD`、`OPENMONTAGE_ALLOWED_PROVIDERS`  
- Key 名见 [配置/00-字段清单.md](../配置/00-字段清单.md)

### 选择有哪些

| 选择 | 说明 |
|------|------|
| 重度 | 付费 TTS + 生图 + 生视频（门禁：估价 → 样片 → 正式） |
| 中度 + 付费 TTS | 须你主动说「用云端 TTS」；画面仍可走 Stock |
| 白名单商 | 图/视频/TTS 商列表由 Skill03 与配置清单列出 |

### 对应 Skill / MCP

| 能力 | Skill | MCP |
|------|--------|-----|
| 填 Key 引导 | `openmontage-bootstrap-providers`（Skill03） | — |
| 付费语音 | `openmontage-providers-tts` | `openmontage-providers-tts` |
| 付费生图 | `openmontage-providers-image` | `openmontage-providers-image` |
| 付费生视频 | `openmontage-providers-video` | `openmontage-providers-video` |

produce **编排**付费步骤，**不静默**调付费 API。失败不静默换商。

Skill03 示例：

> 按 Skill03：给 Kling 填 Key，只口述改 env，先别调用付费 API。

## 对照一览

| | 免费路径 | 收费路径 |
|--|----------|----------|
| 默认 MCP | 门面 + Stock（Key 可空） | 三个 providers（Key 后填） |
| 默认档位 | 轻度；中度用 Stock | 重度；或中度点名付费 TTS |
| 钱 | $0（注意 API 限速） | 云厂商计费 + 门禁确认 |
