# 03 — 字幕与配乐

> Skill：`openmontage-bootstrap-captions-music`（门面 `produce_*`）  
> 状态：文稿→字幕、BGM 登记与 compose 输入已可用；付费音乐 / 字幕时间位置未做。

## 何时用

- 需要字幕轨，和/或本地 BGM  
- 旁白仍走 produce TTS（本 Skill 不强制旁白）

## 默认约定

| 项 | 约定 |
|----|------|
| 无文稿 | Agent 起草短脚本 → 人审 → 切分字幕 |
| 配乐 | 默认本地/零 Key；付费音乐须显式选择（未落地） |
| 目录 | `assets/copy/` · `music/` · `subs/` · `audio/` |

## 流程摘要

```text
scan / ensure dirs
→ 写或导入文稿 → segment_copy_to_subtitles(confirm)
→（可选）import/register_music
→ produce_build_compose_inputs → 交回 Skill02 compose
→（可选）produce_mix_narration_and_music（需 ffmpeg）
```

细节与工具名以仓内 Skill 为准；编排交接见 `openmontage-bootstrap-produce`。
