# 01 — 安装、配置与环境检测

> 前提：已有 OpenClaw。细节由 Skill 执行；本文只给入口与清单。

## 安装闭环（推荐顺序）

```text
① OpenClaw 放入并启用外置 installer（大更新后须再从仓内覆盖拷贝）
② 无仓库：clone｜已有：git pull（用户确认后 Agent 可代跑）
③ 一次性注册 5 个 MCP + extraDirs + 启用 5 个 Skill（Key 可空）
④ 闭环检查通过 → setup → verify_ready → 可出片
```

仓内 Skill 靠 `extraDirs` 随 `git pull` 更新；**仅 installer 需外置再拷贝**。

## 对应 Skill

| 场景 | Skill |
|------|-------|
| 新装 / 已有仓库更新 | `openmontage-bootstrap-installer`（外置拷贝） |

| 错误处理 | `openmontage-bootstrap-error-handling` |
| 字幕配乐 | `openmontage-bootstrap-captions-music` |
