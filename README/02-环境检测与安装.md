# 02 — 环境检测与安装（Skill01）

使用 Skill：`openmontage-bootstrap-setup`  
MCP：`openmontage-bootstrap`

## 协议（必须）

1. `detect_environment` — 看缺口  
2. `plan_install` — **向用户展示完整改动计划**（不要直接装）  
3. 用户明确同意后，再对每步调用：`dry_run=false` + `confirm_execute=true`  
   - `install_python_deps`  
   - `install_node_deps`  
   - `ensure_ffmpeg`  
   - `ensure_piper_model`  
   - `configure_sandbox`  
4. 若返回 `manual_commands` / `skipped_no_admin_or_failed`：把命令交给用户手动执行，不要假装成功  
5. `verify_ready` — 仅当 `ready_for_skill02` / `can_produce_video_now` 为真时进入出片  

## 试跑提示词示例

> 先检测环境，给我看完整安装计划，不要直接改系统。

确认后再说：

> 按计划执行安装。
