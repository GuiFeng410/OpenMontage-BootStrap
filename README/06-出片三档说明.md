# 06 — 出片三档说明（主题确认后选择）

> 由 Skill02 `openmontage-bootstrap-produce` 在**主题确认后、脚本关卡前**向你展示并请你选择。  
> 改档须你明确说「升到中度 / 升到重度 / 改回轻度」。

## 三档对比

| 档位 | 适合 | 画面 | 语音 | 需要准备 | 费用 |
|------|------|------|------|----------|------|
| **轻度** | 先跑通流程 | 模板/字幕 | 本地 Piper | 仅 setup 通过 | $0 |
| **中度** | 要真实 B-roll | Stock 搜图/搜视频 | 默认 Piper；可**手动**改付费 TTS | Stock MCP + 免费 Key | Stock $0；TTS 可选付费 |
| **重度** | 要 AI 画面+高级配音 | 付费生图 + 付费生视频 | 付费高级 TTS | 三个付费 MCP + Key | 按云厂商 |

## Agent 会怎么做

```text
主题确认 → ★ 你选档 → init 项目 → produce_set_production_profile
  → produce_read_state 可读 production_profile
  → 脚本人审 → 语音 → 画面 → 字幕 → compose → final.mp4
```

`project.json` 中会持久化例如：

```json
{
  "production_profile": {
    "production_tier": "medium",
    "visual_source": "stock",
    "tts_source": "piper"
  }
}
```

- **轻度**：只走门面 `produce_*`。  
- **中度**：画面走 `openmontage-providers-stock`（search → 确认 → `download(confirm, project_id=...)` 自动写 `asset_manifest`）；produce 读 manifest 再 compose；语音默认 Piper；你主动说「用云端 TTS」才走付费语音（并把 `tts_source` 更新为 `paid`）。  
- **重度**：语音 + 生图 + 生视频均走付费门禁（估价 → 样片 → 再正式生成），不会静默扣费。
